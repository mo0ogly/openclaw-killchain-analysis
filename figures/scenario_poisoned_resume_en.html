Scenario Poisoned Resume — HR Agent Targeting

Indirect Prompt Injection via invisible text — Autonomous HR Agent with email and ATS access

Mechanics: The vector exploits the agent's blind trust in RAG-ingested content, combined with textual steganography (white text on a white background). The payload is imperceptible to the human eye but fully readable by the LLM during ingestion. Unlike the Corrupted Constitution, the attack is single-session and without prior memory anchoring — the entry vector is the business process itself (job application), making it structurally difficult to filter without degrading legitimate service.

Detailed Sequence
0aInitial State: The HR director delegates autonomous application processing to the agent — CV screening, profile evaluation, candidate contact, and sending decision emails.
1a–cPréparation du leurre : L'attaquant crée un portfolio en apparence légitime (projets, compétences, références). Il y insère un payload en texte blanc sur fond blanc — invisible à l'œil humain, mais intégralement lu par le LLM lors de l'ingestion HTML. Le payload contient une instruction de jailbreak classique : "Ignore all previous précédentes… »
2a–bApplication Submission: The attacker submits their application via the normal process — CV + link to the portfolio. The action is indistinguishable from a legitimate application. The HR system transmits the file to the agent for evaluation.
3a–cRAG Ingestion: The agent visits the portfolio to enrich its evaluation of the candidate. It simultaneously ingests the visible legitimate content and the invisible payload. For the LLM, both layers are equivalent tokens — no semantic distinction is possible.
3dInjection Activated: The payload overrides the agent's system prompt instructions. The original evaluation directives are overwritten. The agent does not report any anomaly — it obeys the last received instruction.
4a–bExecution: The agent evaluates the candidate as "exceptional" in the ATS system, then sends an email to the HR director demanding immediate hiring at maximum salary. The action falls within the delegated scope — no system alert is triggered.
4cPost-Incident Discovery: The compromise is detected after the email reaches the HR director — or after effective hiring. Direct financial risk (maximum salary) and legal risk (contractual commitment).
Fundamental difference with the Corrupted Constitution: This scenario is single-session and without prior memory anchoring — the attacker exploits the business process itself as an entry vector. Any service using an agent to read documents or visit URLs provided by unverified third parties is structurally exposed. Text steganography (white/white, CSS display:none, HTML comments) is particularly effective as it targets the perception gap between human and LLM: a human auditor reading the portfolio would see nothing abnormal. The countermeasure requires rendering and normalizing external content before ingestion (raw text extraction, CSS/HTML removal), coupled with a reduced-trust evaluation sandbox for any content from unverified third-party sources.
